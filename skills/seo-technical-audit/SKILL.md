# SEO Technical Audit (Full Stack Workflow)

Этот навык объединяет классический технический сканер и аналитические способности LLM для создания глубокого отчета.

## Возможности

1.  **Технический скан:** Коды ответов, метатеги, заголовки H1-H4, проверка изображений (Alt), mixed content.
2.  **Инфраструктурный чек:** Проверка наличия и доступности robots.txt и sitemap.xml.
3.  **Lighthouse Core Web Vitals:** (Опционально) Замер LCP, CLS, FCP через консольный Lighthouse.
4.  **AI Анализ:** Качественная оценка логики заголовков, релевантности анкоров и чистоты структуры.

## Процесс работы

### Шаг 1: Инициализация
Агент запрашивает URL сайта. Если пользователь не указал протокол, по умолчанию используется `https://`.

### Шаг 2: Автоматический сбор данных
Агент запускает внутренний инструмент `audit_tool.py`:
```bash
python3 ~/.openclaw/workspace/skills/seo-technical-audit/audit_tool.py <URL> [--lighthouse]
```

### Шаг 3: Качественный AI Анализ
После получения JSON-отчета, агент проводит смысловой аудит:
- **Логика H1-H4:** Нет ли пропусков уровней? Соответствуют ли заголовки тематике страницы?
- **Метаданные:** Насколько Title и Description привлекательны (CTR) и содержат ли ключи?
- **Контент:** Поиск "битых" паттернов или отсутствия важной семантики.

### Шаг 4: Формирование финального отчета
Отчет делится на блоки:
1.  **Сводка (Score):** Общая оценка состояния.
2.  **Критические ошибки (Critical):** То, что нужно исправить немедленно (4xx/5xx, noindex, mixed content).
3.  **Предупреждения (Warnings):** Отсутствие Alt, длинные метатеги, пропуски H-заголовков.
4.  **AI Рекомендации:** Советы по улучшению контента и структуры.

## Использование

Для запуска аудита достаточно отправить сообщение:
`Проведи технический аудит https://example.com`

Если нужны показатели скорости загрузки (занимает больше времени):
`Проведи полный технический аудит https://example.com с Lighthouse`
