# 04. Шаблон карты сущности бренда

> Карта сущности — каталог всех площадок, где бренд присутствует или должен присутствовать, с текущими данными, статусом и списком правок. Базовый GEO/AEO-инструмент: обеспечивает единообразие данных (NAP) во внешнем контуре и «понятность» бренда для AI-систем.
>
> Обычно ведётся как Google-таблица. Референс: карта сущности **ЛаВояж** (`КартаСущностиЛаВояж.html`). Пока делаем **вручную** (см. [[03_Распределение работ и роли]] — ассистенты + SEO-спец).

---

## Столбцы (фиксированные)

| Столбец | Что вносим |
|---------|-----------|
| Источник | название площадки/тип (Официальный сайт, Яндекс, 2ГИС, Zoon…) |
| URL | ссылка на карточку/профиль |
| Название | как компания названа на площадке (сверяем на единообразие) |
| Телефон | контактный телефон |
| Режим работы | часы (офис/инфоцентр) |
| Адрес | юр./фактический адрес |
| Сайт | ссылка на офиц. сайт |
| Почта | e-mail |
| Telegram / ВК / ОК / RuTube / Max | ссылки на каналы (столбец на канал) |
| Статус | `Согласовать правки` (карточка есть) / `Создать карточку` (нет) |
| Комментарий | что добавить/исправить (услуги, прайс, фото офиса/команды, сертификаты, витрины, ответы на отзывы, ссылки на соцсети) |
| Рейтинг | текущий рейтинг и число оценок |

---

## Строки — чек-лист площадок

### 🟦 Универсальные (для любого проекта)
- **Официальный сайт** — эталон данных, с которым сверяем все остальные.
- **Карты:** Яндекс Карты, 2ГИС, Google Business Profile.
- **Соцсети/каналы:** VK, OK, Telegram, RuTube, Max (что есть у клиента).
- **Отзовики/репутация:** Zoon, otzovik.online, otzyv.media, irate.info, negativ.info, otzovikon.com, otzyv.pro, otzyv-pro.ru, eto-razvod.ru, sdelaivibor.ru.

### 🟨 Отраслевые (набор под нишу)
Пример для travel (ЛаВояж): РСТ (Российский союз туриндустрии), Profi.Travel, 1001тур, Каталог турфирм России, Турпомощь.
> Под другую нишу — свой набор отраслевых справочников и рейтингов (подбирается на старте).

---

## Логика работы с картой

1. **Инвентаризация:** пройти по всем площадкам, зафиксировать URL, данные и рейтинг (или отметить `Создать карточку`).
2. **Сверка (NAP):** название/адрес/телефон/сайт/режим — привести к единому виду по эталону сайта.
3. **Правки:** заполнить «Комментарий» — что добавить (услуги, прайс, фото офиса и команды, сертификаты, витрины, ссылки на соцсети, ответы на отзывы).
4. **Согласование:** статусы `Согласовать правки` — согласовать с клиентом (точка касания).
5. **Внедрение:** правки/создание карточек (ассистенты), импорт в Bing Места из Google Business.
6. **Поддержка:** отзывы по согласованной стратегии, мониторинг тональности.
