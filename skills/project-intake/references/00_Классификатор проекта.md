# 00. Классификатор проекта Artkod

> Назначение: на входе разметить проект по 6 осям. Как только все оси заполнены — тип проекта определён, а дальше по [[01_Матрица маршрутизации]] известно на 80%: кто ведёт, по какому пайплайну, какие этапы, дедлайны, KPI и тариф.
>
> Как пользоваться: заполнить блок «Карточка проекта» ниже (копируется под каждого клиента), затем открыть матрицу и найти строку под получившийся тип.

---

## 6 осей разметки

### Ось A. Услуга / направление
Что клиент реально покупает. Может быть одна или комбинация.

| Код | Услуга | Комментарий |
|-----|--------|-------------|
| `SEO` | Классическое SEO | позиции/трафик в Яндекс/Google |
| `GEO` | GEO/AEO | продвижение в нейросетевых выдачах (ИИ-ответы) |
| `MAPS` | Геосервисы | Яндекс.Карты, 2ГИС, Zoon |
| `DEV` | Разработка | сайт/лендинг/доработки (WordPress) |
| `PPC` | Директ / контекст | Яндекс.Директ |
| `SERM` | Репутация | SERM/ORM, отзывы |
| `MIX` | Комплекс | несколько направлений в одном договоре |

### Ось B. Тип бизнеса клиента
Определяет тональность, семантику, тип страниц, кейсы для КП.

| Код | Тип | Примеры клиентов |
|-----|-----|------------------|
| `B2B-PROD` | B2B-производство / опт | Элимет (фурнитура), Стальтек (металл), Сталь-Инвест |
| `LOCAL` | Локальный бизнес (есть точки/гео) | Мало Места, Мебель-Москва |
| `ECOM` | Интернет-магазин | — |
| `SERVICE` | Услуги / тревел | ЛаВояж (круизы), Штурман |

### Ось C. География
| Код | Охват |
|-----|-------|
| `GEO-1` | 1 регион |
| `GEO-2-3` | 2–3 региона |
| `GEO-RF` | вся РФ / федеральный |
| `GEO-LOCAL` | локальный, привязка к точкам (важно для MAPS) |

### Ось D. Состояние сайта
| Код | Ситуация |
|-----|----------|
| `SITE-NONE` | сайта нет → нужен DEV на старте |
| `SITE-OK` | есть, тех. состояние норм |
| `SITE-BAD` | есть, но нужны правки/редизайн |
| `SITE-OTHER` | есть ограничение по правкам |

### Ось E. Цель и KPI
Что считаем результатом. Обычно 1 главная + 1 сопутствующая.

| Код | Цель / KPI |
|-----|-----------|
| `KPI-POS` | позиции (напр. 60% в ТОП-10 к 4-му мес., 80% к 6-му — как у Элимет) |
| `KPI-TRAF` | рост поискового трафика |
| `KPI-LEAD` | лиды / заявки |
| `KPI-AI` | упоминания и цитирование в нейросетях (GEO/AEO) |
| `KPI-REP` | репутация: рейтинг, отзывы (SERM) |

### Ось F. Тариф / бюджет
| Код | Пакет | Цена/мес |
|-----|-------|----------|
| `T-DIRECT` | Директ базовый | от 45к + бюджет |
| `T-SEO-BASE` | SEO базовый | от 52к |
| `T-GEO-1` | SEO/GEO/AEO пакет 1 | 64к |
| `T-GEO-2` | SEO/GEO/AEO пакет 2 | 112к |
| `T-GEO-3` | SEO/GEO/AEO пакет 3 | 262к |
| `T-MIX` | миксованный | считается индивидуально |
