# Методология «Доказательное SEO 2026» (требования к контенту в ТЗ)

Эти правила внедряются в Фазе 3 (синтез ТЗ) для защиты от фильтра `lowQuality`
(Anti-AI) и попадания в блоки AI Overviews (SGE). Источник — гайд
«Доказательное SEO 2026», лежит в `/raw/AI Агенты/`.

## 1. Content Effort (уникальное усилие)
Запрет на простой пересказ ТОП-10.
**Требование в ТЗ:** «Добавьте минимум 1–2 факта/данных, которых нет ни у одного
конкурента из ТОПа (см. блок Gap в top-map.json): реальный кейс компании, расчёт
стоимости на конкретном примере, скриншот из CRM/сметы, собственный тест/замер».
**Цель:** повышение `contentEffort`.

## 2. Декларативные заголовки (чанкинг для ИИ)
**Требование в ТЗ:** «Заменяйте общие подзаголовки ("Как выбрать") на
декларативные утверждения-факты ("Авторский надзор экономит до 15% бюджета на
переделках"). Каждый H2/H3 — самостоятельный смысловой чанк».
**Цель:** цитируемость в AI Overviews.

## 3. Original Media Score
**Требование в ТЗ:** «Минимум стоковых фото. Каждое изображение — оригинальное:
фото с объекта, уникальная инфографика, скриншот интерфейса/сметы».
**Цель:** подтверждение Experience через `original_media_score`.

## 4. Сигнальная когерентность (Goldmine)
**Требование в ТЗ:** «Строгое смысловое соответствие Title ↔ H1 ↔ первый абзац.
Первый абзац сразу отвечает на интент, без воды и длинных вступлений».
**Цель:** удержание контроля над сниппетом.
Опорный сигнал: `tech.definitionIntroShare` в top-map.json — доля конкурентов,
у кого первый абзац уже даёт прямой ответ. Если высокая — это норма ниши.

## Как методология связана с данными top-map.json
- `volume.recommendedMin/MaxWords` — объём (медиана ТОПа + 10–15%), не «вода».
- `headingMap[].weight` — Mandatory/Recommended/Optional блоки структуры.
- `tech.listMedian / tableMedian / faqShare / schemaTypesUnion` — обязательные
  форматные элементы (списки, таблицы, FAQ, schema), подтверждённые ТОПом.
- `lexicon` — реальные слова/фразы конкурентов для естественного вхождения.
- Gap (блоки нейроструктуры, которых нет в `headingMap`) → пункт 1 (Content Effort).
